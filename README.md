# Udacity Neighbourhood Map

This is the project submission for the Udacity Neighbourhood Map project.
It uses the [Google Maps JavaScript API](https://developers.google.com/maps/documentation/javascript) and the [Foursquare API](https://developer.foursquare.com/).
The color scheme is a modified version of the style by [thomasf](https://thomasf.github.io/solarized-css/).

It uses the JavaScript libraries [knockout.js](http://knockoutjs.com/), [underscore.js](http://underscorejs.org/) and [jQuery](https://jquery.com/).

## Prerequisites

You need to have [yarn](https://yarnpkg.com/en/) installed.

To download and install the JavaScript dependencies use

```yarn install```

## Running the App

Simply open `index.html` in your browser.
